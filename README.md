# jquery-plug-in
plug-in basing on Jquery
<h2>模态框</h2>
<div>
	<p>1.引入css文件（建议复制粘贴至项目css文件中）</p>
	<p>2.引入js文件（建议使用发布的压缩版）</p>
	<p>3.创建模态框对象，并传入参数。</p>
</div>
<h4>参数说明</h4>
<p>cover:true</p>
<p>是否显示遮罩层：ture(有),false(没有)</p>
<p>num:0</p>
<p><i style="color: #f00;">*</i>必填 模态框索引：自定义整形数字（创建多个模态框时用以区分）</p>
<p>	pos:"center"</p>
<p>w:300</p>
<p>h:300</p>
<p>scroll:true</p>
<p>content:true</p>
<p>closeBtn:true</p>
<p>style:"fade"</p>
